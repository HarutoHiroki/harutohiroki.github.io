Inviting Cryptonix X to your server

To invite Cryptonix X to your server, you will need a server which meets some requirements:

  • Active server
  • Non-abusive members (aka. people who will spam bots*)
  • At least 15 members (doesn't count bots)
  • No more than 6 bots (including Cryptonix X)

* note: Since this bot has an active bug testing program where users can be rewarded according to the bug they found, your server need to agree to report the bug the moment you found it and not to spam the bug so the bot crashes.

If you want to take further contact, visit my Discord server by clicking the Discord button above, then talk to me at HarutoHiroki#4000. Also remember, the settings are in JSON format, so I hope you have some experience with it.
