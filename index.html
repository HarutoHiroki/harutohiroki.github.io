About me

Sound

Hey, Haruto here

I'm a freelance Full-Stack Developer with a passion for Virtual Reality and a deep devotion to the world of Music. With over 6 years of coding experience, I've worked on many projects, mainly specializing in back-end development. My recent venture into front-end design has been a thrilling expansion of my skill set, culminating in the creation of this website. It's a continuous work in progress, a project close to my heart, where I blend my comprehensive expertise in both back-end and front-end.

My Skills

My toolkit is diverse, encompassing languages like HTML, CSS, JavaScript, Node.js, and Java. I leverage these in various projects, using tools like Microsoft VS Code, Google Cloud Platform, and AWS to bring ideas to life. My journey also includes exploring hardware design principles and delving into microcontrollers like Arduino, Raspberry Pi, and ESP32.

My Interests

My world revolves around the thrill of Programming and the unique satisfaction in debugging. My passion for Music, with a special fondness for the rhythmic allure of J-Pop, the rich textures of orchestral compositions, and the storytelling in original soundtracks, is a constant source of inspiration. Beyond these melodies, my curiosity stretches into the intricate world of In-Ear Monitors and the immersive realm of Virtual Reality, where Iwhere I explore both the technology behind VR and the captivating experiences it offers.

Projects

> Haruto's Graph Tool, my version of the CrinGraph public repository, customized for the real-time comparison and creation of equalization profiles for headphones and in-ear monitors.

> Haruto's Graph Extension, a Manifest V2 browser extension that enables the usage of any supported GraphTool's expansive EQ function on any website with a single click.

> Paper++, a program that converts user-submitted, handwritten code to correctly formatted Java files, running them and returning the output.

> Cryptonix X, a multi-purpose Moderation, Utility, and Image Generation Discord bot, written in JavaScript using the Discord.js library.

> alexflipnote.js, a lightweight and user-friendly JavaScript wrapper for the Image Generation API by AlexFlipnote.

> Discord.js Bot, a project designed to assist beginners in coding Discord.js bots.
